超便利なんですケド！

それはchromecastでございますです(^^)

Google Chromecast ( クロームキャスト ) Wi-Fi経由 テレビ接続 ストリーミング / HDMI / 802.11b/g/n / 1080p ) GA3A00035A16


Amazonで詳しく見る


コイツ、超便利すぎてやばいです。

うちのテレビは東芝のレグザの古いやつなんですが、HDMI端子にぶっさして、電源用にUSBをさすだけ。あとはiPadやwindowsのChromeからキャストするだけ。

妖怪ウォッチ第一体操（？）をテレビにキャストすると子どもたちが大喜びしております（子供がよろこぶ使い方）。

その他にはDailymotionなどのサイトの動画も720pでキャストできるので妻も私も大喜びしております（大人が喜ぶ使い方）。

※著作権などには気をつけましょう

Google changes the worldって感じです。
